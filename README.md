# NEB GPA Calculator

Live site: https://neb.amritkc.com.np

A free GPA calculator for NEB students in Nepal.

## Structure

- `index.html` — landing page
- `pages/science.html` — Science GPA calculator
- `pages/management.html` — Management GPA calculator
- `pages/see.html` — SEE GPA calculator
- `assets/css/main.css` — global landing page styles
- `assets/css/calculator.css` — calculator page styles
- `assets/js/main.js` — landing page interactions
- `assets/js/calculator.js` — shared GPA calculation logic
- `assets/images/` — image assets
